# สูตรคำนวณผลการประเมิน

|-----------|---------------|

| 1.51 - 2.50 | น้อย (Poor) |
| 1.00 - 1.50 | น้อยที่สุด (Very Poor) |

## สูตรคำนวณ
- **ค่าเฉลี่ย (Mean)**: x̄ = ΣX / N
- **ส่วนเบี่ยงเบนมาตรฐาน (S.D.)**: S.D. = √[Σ(X - x̄)² / (N-1)]

โดยที่:
- X = คะแนนที่ได้
- N = จำนวนผู้ตอบแบบสอบถาม
- x̄ = ค่าเฉลี่ย
